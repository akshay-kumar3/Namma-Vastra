Namma Vastra

Namma Vastra is an Android application developed using Kotlin, Jetpack Compose, and Firebase Firestore to support Karnataka’s traditional handloom weavers.

Features

* Saree Gallery
* Trending Sarees
* Material Price Calculator
* Weaver Story Section
* WhatsApp Inquiry Feature
* Firebase Firestore Integration

Technologies Used

* Kotlin
* Android Studio
* Jetpack Compose
* Firebase Firestore
* Material UI Design

Purpose

The application helps local weavers promote and sell traditional sarees digitally without middlemen while preserving Karnataka’s handloom culture.
